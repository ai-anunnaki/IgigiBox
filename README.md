# 鸬鹚工具箱 LuciBox

macOS系统工具箱应用

## 功能规划

### ✅ 第一期：进程管理
- 查看系统进程（PID、进程名称、端口号）
- 杀掉指定进程
- 实时刷新进程列表

### 🔜 第二期：文件管理
- 浏览系统文件夹
- 文件增删改查
- 复制、移动、重命名

### 🔜 第三期：剪贴板管理
- 查看当前剪贴板内容
- 查看历史复制记录

## 技术栈

- Swift 5.9+
- SwiftUI
- macOS 13.0+

## 开发

```bash
# 在Xcode中打开项目
open LuciBox.xcodeproj

# 或使用命令行构建
xcodebuild -project LuciBox.xcodeproj -scheme LuciBox -configuration Debug
```
